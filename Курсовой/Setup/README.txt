Вітаю! Для установки програми необхідно запустити файл setup.exe(\Setup\Debug\setup.exe), після цього встановити програму.
Після установки програми, необхідно виконати наступні кроки, щоб вільно користуватися програмою:
-Спершу необхідно встановити MySqlConnector(\Setup\MAMP+SQLCONNECTOR\MySqlConnector\SetupMySqlConnector.msi)
-Далі встановити MAMP(\Setup\MAMP+SQLCONNECTOR\MAMP\SetupMamp.exe)
-Після цього запустити MAMP та зайти в phpMyAdmin, де необхідно імпортувати базу даних(\Setup\БД\items.sql)
